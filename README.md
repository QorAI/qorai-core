# Qorai Core

Qorai Core is a set of changes, APIs, and scripts used for customizing Chromium to make the Qorai browser. Please also check https://github.com/qorai/qorai-browser

Follow [@qorai](https://twitter.com/qora_net) on Twitter for important announcements.
